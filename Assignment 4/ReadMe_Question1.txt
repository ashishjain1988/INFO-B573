1. The R script file for question one is question_1_4.r and the genomic sequence file is U00096.ptt.
2. Put both the script and the genomic sequence file in the same folder.
3. Now run the R console by typing R and hitting enter in the same directory.
4. After that, run the script by typing command "source(paste(getwd(),"/question_1_4.r",sep=""))".
5. After that quit the R command prompt by typing q().
6. You can see the resultant histogram in Rplots.pdf file in the same folder.
